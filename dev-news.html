Development News!

4/8/19

Some more development today! A little news!

  • The game over notice now disappears when you start a new level!
  • Fixed a bug where the game was calculating the score as many times as there were squares matched - giving you triple or more the score you earned! Scoring is now considerably less generous
  • The project is now on Github!
Next up:
  • Responsive text sizing, so the game looks right on various screen sizes
And after that:
  • Setting the size of images on the buttons according to the screen size, so the game looks better on different screen sizes
  • Some prettification of the scoreboard so it looks the way I want
  • A refactor to make the code run a bit more efficiently
  • Another refactor to prepare me for being able to generate different levels

3/8/19

It's been a hellish year, but I'm finally back.

Before I stopped coding, I tried to add pictures to the squares and broke it, then ran out of spoons in spectacular fashion. I've been back for a few days, fixing bugs and making the game work again. Happy news: it does! So, new features:

  • A restart button that restarts the game!
  • Pretty pictures!
  • Some bug fixes!
Next up for me:
  • Making responsiveness work better (or, really, at all) so you can play on mobile
Also planned:
  • Some more design, to make the site a bit prettier (especially the scoreboard)
  • Levels and targets of varying kinds
  • Playing with difficulty levels in some way
  • Special squares
  • And more!
As you see, I'm full of plans! Development is likely to go in bursts, because I'm that kind of person. But I'm here for now and this is slowly improving!
